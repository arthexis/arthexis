> Maintenance note: every PR that introduces a new admin UI component (or changes an existing component contract) must update this index in the same change.

## Additional token-only consumers

The following first-party admin template currently uses `admin-ui-*` design tokens in local styles without consuming the shared component block classes above:

- `apps/ops/templates/admin/ops/operator_journey_step.html`

When refactoring token-only consumers, prefer migrating to the shared contracts in this index before adding new local class systems.
