# 1.0 decision record

Status: accepted (planning baseline for `1.0.0`).
Date: 2026-03-27.

## Active runtime apps to keep

For `1.0.0`, runtime app wiring keeps:

- `PROJECT_APPS`
- `DJANGO_CORE_APPS`
- `THIRD_PARTY_APPS`
- `PROJECT_LOCAL_APPS`

from `config/settings/apps.py`, with legacy shims removed by the major-version gate.

## Legacy apps to drop

For `1.0.0`, drop all entries from `LEGACY_MIGRATION_APPS` from runtime app loading:

## Upgrade stance for 1.0

`1.0.0` is defined as a fresh-install/import baseline reset. In-place upgrade from existing `0.x` databases is not supported after this reset lands.
