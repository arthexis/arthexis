# Optional features

These docs cover optional suite and node features that can be enabled per deployment instead of suite-wide baseline architecture.

- [LLM LCD log summary](llm-summary.md) — optional `llm-summary-suite` gate plus `llm-summary` node feature for Control-node LCD log summaries.
- [Node features](node-features.md) — capability catalog, role defaults, eligibility checks, and manual enablement workflow.
- [Self-service token management](token-management.md) — scoped token issuance, one-time secret reveal, and audited revoke/rotate controls for operator roles.
