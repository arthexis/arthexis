WASM Net Monitor

Decommissioned

Pyxel viewport removed

The Pyxel-based WASM viewport was retired as part of the 0.2.0 release preparation. Deployments no longer need to stage the generated assets inside static/ocpp/net_monitor/pyxel_viewport.

Continue using the net monitor console without the Pyxel viewport or integrate an alternate visualization pipeline that matches your deployment needs.

Note: Pyxel launcher scripts and assets are no longer bundled. Remove any automation that depended on pyxel_viewport outputs.
